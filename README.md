# Pisano Task

- A market application where products are listed
- You can access the details of each item by clicking on it.

## Requirements

- Deployment Target : iOS 13.0
- xCode Version : 13.3

## Technologies

- Snapkit (Programmatically Design)
- SPM
- Alamofire
- KingFisher
- Network
- MVVM
